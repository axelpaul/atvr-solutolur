# atvr-solutolur

Sales figures (sölutölur) from [ÁTVR](https://www.vinbudin.is/) — Vínbúðin, the
Icelandic state alcohol and tobacco monopoly. The original reports are
published as Excel workbooks.

## Layout

```
YYYY/
  YYYY-MM-month.xlsx   monthly workbook
  YYYY-annual.xlsx     full-year workbook
```

- Annual files are available from 2015 onward (2015 covers 2011–2015).
- Monthly files are available from 2023 onward.
- Month names are in Icelandic (`janúar`, `febrúar`, …).
